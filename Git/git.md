## 파일이나 폴더 올리기
  1. 로컬 저장소로 이동
  2. git add (파일이나 폴더)
  3. git commit -m "메세지" - 커밋
  4. git push origin master - 푸쉬
  
## 폴더 삭제
  1. cmd나 깃배쉬 이용하여 로컬 저장소로 이동 ( cd C:\Users\jh\git\Prepare ) - 경로
  2. git rm -rf ( 삭제 하고싶은 파일이나 폴더 이름 ) - 삭제
  3. git commit -m "커밋메세지" - 커밋
  4. git push -u origin master - 푸쉬
